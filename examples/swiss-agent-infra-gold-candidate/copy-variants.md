## 开头备选

### 更技术

Agent 基础设施这半年最明显的变化，是讨论重心从模型能力转向运行时、权限、上下文和部署系统。

### 更小红书

如果你最近只盯模型榜，可能会错过 Agent 更大的变化。

### 更产品

企业落地 Agent 时，真正难的往往不是“模型会不会”，而是“系统接不接得住”。

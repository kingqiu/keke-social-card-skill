# QA

## Golden Example Status

Status: `Pending Assets`

This folder is an intake scaffold for the first AI tool launch candidate. It is not a rendered example and must not be used as an aesthetic reference.

## Required Before Rendering

- Real product/Skill screenshots or output images are supplied.
- `assets/IMAGE_REQUESTS.md` maps each image role to a page.
- `assets/SOURCES.md` records user-supplied, generated, or public-source assets.
- Page plan is updated from the actual product promise.
- Visual system is confirmed as Rednote Native + Proof Lab.

## Gold Upgrade Gate

Upgrade only after:

- rendered images pass validator with 0 fails;
- first card has an immediate reason to click;
- proof/result areas are inspectable;
- page 2+ match cover quality;
- manual visual review finds no embarrassing page.

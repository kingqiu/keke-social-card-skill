# Font Sources

Bundled fonts are open-source fonts served by Google Fonts.

| Family | Files | License |
|---|---|---|
| Inter | `keke-inter/*.ttf` | SIL Open Font License 1.1 |
| Noto Sans SC | `keke-noto-sans-sc/*.ttf` | SIL Open Font License 1.1 |
| IBM Plex Mono | `keke-ibm-plex-mono/*.ttf` | SIL Open Font License 1.1 |
| Noto Serif SC | `keke-noto-serif-sc/*.ttf` | SIL Open Font License 1.1 |
| Playfair Display | `keke-playfair-display/*.ttf` | SIL Open Font License 1.1 |

These files are vendored so card rendering stays stable without network font loading.
